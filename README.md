# 📖 个人第二大脑

欢迎来到您的知识管理系统。这个vault旨在帮助您组织和连接关于健康、目标、学习和项目的所有信息。

## 🗂️ 结构概览

### 核心文件夹

- **[[projects/]]** — 您正在进行的项目和倡议
- **[[health/]]** — 健康跟踪、习惯、锻炼、营养
- **[[goals/]]** — 短期和长期目标、里程碑、回顾
- **[[learning/]]** — 学习笔记、课程、技能发展

### 知识库

- **[[entities/]]** — 人物、工具、资源、地点
- **[[concepts/]]** — 想法、原则、框架、方法论

### 系统文件夹

- **[[.raw/]]** — 待处理的原始内容（剪藏、笔记、转录）
- **[[_templates/]]** — 笔记模板
- **[[meta/]]** — 索引、仪表板、系统文档

## 🚀 快速开始

1. **捕获** — 将想法、文章、笔记放入 `.raw/` 文件夹
2. **处理** — 提取实体和概念，创建连接
3. **组织** — 移动到适当的文件夹，添加frontmatter
4. **连接** — 使用 `[[双链]]` 连接相关笔记
5. **回顾** — 定期查看 [[meta/dashboard|仪表板]]

## 📊 仪表板

查看 [[meta/dashboard|📊 仪表板]] 获取您的vault概览、最近活动和关键指标。
