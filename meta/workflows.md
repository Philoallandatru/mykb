# 🔄 工作流程

## 📥 每日捕获

1. 使用Obsidian快速捕获或Web Clipper将想法保存到 `.raw/`
2. 每天或每周安排时间处理收件箱
3. 为重要内容创建permanent笔记

## 📝 处理原始内容

1. 打开 `.raw/` 中的文件
2. 识别关键实体（人物、工具、资源）
3. 识别关键概念（想法、原则、框架）
4. 为每个创建单独的笔记
5. 使用 `[[双链]]` 连接相关内容
6. 添加适当的frontmatter和标签
7. 归档或删除原始文件

## 🎯 目标管理

1. 在 `goals/` 中创建新目标
2. 定义成功标准和行动步骤
3. 每周回顾进度
4. 将目标链接到相关项目
5. 庆祝完成的目标

## 📁 项目管理

1. 在 `projects/` 中创建新项目
2. 定义目标和任务清单
3. 链接到相关的学习资源和目标
4. 定期更新状态和进度
5. 完成后记录成果和学习

## 📚 学习跟踪

1. 在 `learning/` 中创建学习笔记
2. 边学边记录关键要点
3. 创建永久笔记的概念
4. 制定实践任务
5. 定期回顾和应用

## 💪 健康跟踪

1. 使用模板记录锻炼、营养、习惯
2. 跟踪指标和进度
3. 识别模式和趋势
4. 根据需要调整
